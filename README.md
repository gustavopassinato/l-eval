## API de avaliação de linguágens de programação

 Esta aplicação foi desenvolvida na edição de 2022 da Imersão Java idealizada pela [Alura](https://www.alura.com.br/) visando o desenvolvimento de uma aplicação utilizando as técnicas mais atuais no mercado.

 ## Contexto

 Esta é a segunda aplicação desenvolvida durante o evento, pois a primeira se constituiu como uma consumidora de API's de terceiros, então para fechamento do projeto desenvolvemos nosso próprio serviço. 

 O objetivo desta API é fornecer uma lista de linguagens de programação, e seus respectivos logos (obtidos principalmente no repositório: [programming-languages-logos](https://github.com/abrahamcalf/programming-languages-logos)). Bem como permitir o registro de novas linguagens a esta lista, alteração de linguagens já inseridas assim como a exclusão de linguagens da lista, ou seja, o CRUD de todos os elementos do banco. 

## Roadmap de desenvolvimento

Com a proposição do desafio, iniciei o desenvolvimento de acordo com o proposto na aula. As escolhas básicas foram: 

* O ecossistema [Spring](https://spring.io/) para sustentar a aplicação;
* O [MongoDB](https://www.mongodb.com/cloud/atlas/register) para fornecer o ambiente de persistência dos dados, sendo utilizada uma versão do banco hosteada em cloud;
* O [Postman](https://www.postman.com/) como ambiente para teste dos endpoints;
* O [Heroku](https://www.heroku.com
) como infraestrutura cloud para publicação da versão final do projeto.
